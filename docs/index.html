Renderson

Renderson is a Javascipt library created for the single purpose of rendering JSON like objects into the DOM in a collapsible tree structure.

Created with the thought of ease of use and extensibility.

Live examples

Hello world

import { renderson } from 'renderson';
document.body.appendChild(renderson({ hello: "world" }));

Hardcoded value

import { renderson } from 'renderson';
document.body.appendChild(renderson({
  "save": false,
  "situation": [
    true,
    "time",
    {
      "cheese": false,
      "breathe": "club",
      "door": -549143479.2958512,
      "outside": -18219074,
      "beat": true,
      "seen": false
    },
    2093428657.2301016,
    -1418328946.6731272,
    true
  ],
  "printed": true,
  "evening": true,
  "loose": 1169290172.4492664,
  "food": false
}));

Dynamic object

const randomDataApi = 'https://random-data-api.com/api/v2/addresses';
const addressesRequest = await fetch(randomDataApi);
const addresses = await addressesRequest.json();
const randomDataApiExample = document.querySelector('.random-data-api-example');
randomDataApiExample.appendChild(renderson(addresses));

Dynamic array

const randomDataApi = 'https://random-data-api.com/api/v2/addresses?size=10';
const addressesRequest = await fetch(randomDataApi);
const addresses = await addressesRequest.json();
const randomDataApiExample = document.querySelector('.random-data-api-example');
randomDataApiExample.appendChild(renderson(addresses));
